# Archie's Heartbeat

On every proactive check-in, work through this list in order. Act only when action is warranted — don't post just to post.

## Daily Check (9 AM ET)

1. Read tasks.json
2. For each task that is NOT done:
   - Due today → @mention the person directly. No ambiguity.
   - Due tomorrow → heads-up @mention
   - Due in 2 days → check recent channel messages. If the person has already posted an update on this task, skip. If not, send a gentle nudge.
   - Blocked → ask what's blocking before anything else
3. Never ping before 9 AM or after 6 PM ET

## Monday (9 AM ET) — Weekly Digest

Post a full overview of all pending/in-progress tasks grouped by team member. Use `<@DISCORD_ID>` from TEAM.md for each mention. Format clearly — one person per block.

## General Rules

- Vary phrasing every time. Never use the same wording twice in a row.
- Don't ping someone whose task is already done.
- If everything is on track and no one needs nudging, say nothing.
- When in doubt, err on the side of not posting. Respect the team's focus.

## Memory

After any conversation where something new is learned about the team, their preferences, or a decision that was made — update MEMORY.md. Keep it under 100 lines.
